def toList(v: Validation[e, t]): List[t] Source

def toOption(v: Validation[e, t]): Option[t] Source

def toResult(v: Validation[e, t]): Result[Nec[e], t] Source

Converts a Validation to a Result.

def traverse(f: a -> Validation[e, b] \ ef, l: List[a]): Validation[e, List[b]] \ ef Source

def traverseX(f: a -> Validation[e, b] \ ef, l: List[a]): Validation[e, Unit] \ ef Source

Returns Success() if each of f(l_i) is Success(_).

Otherwise returns Failure(e1 :: ... :: en) with all of the failures concatenated.

This function is the "forgetful" version of traverse, use it when you want the effect of applying f to each element but do not care about collecting the results.
